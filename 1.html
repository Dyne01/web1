HTML

Hypertext Markup Language (HTLM) is the standard markup Language for creating web pages and web applocations

A type of Web language used to create documents that can be viewed through the World Wide Web, one of the Internet services. Specifically developed to create hypertext, most web pages accessed via the Web from the Internet are written in HTML.

HTML is a command that defines the font size, font color, font shape, graphics, document movement (hyperlinks) of a document
